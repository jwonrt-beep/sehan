# 세한메카트로닉스 설치사례 템플릿 명령어

> 홈페이지 핵심 콘텐츠인 설치사례 페이지에 적용할 **강제 형식** 명령어

---

## 1. 설치사례의 역할

- 홈페이지의 **핵심 콘텐츠**
- "실제로 해봤다"는 신뢰 증명
- 제품 나열이 아니라 **문제 해결 스토리**

---

## 2. 사용 금지

| 금지 | 대신 |
|------|------|
| 로봇 브랜드·모델 중심 서술 | 공정·문제·해결 중심 |
| 기술 스펙 나열 | 현장에서 달라진 점 |
| 홍보 영상 위주 | 현장 동작 확인용 영상 |
| 사진 많이 나열 | 공정 이해가 되는 사진만 선별 |

---

## 3. 강제 5항목 구조

모든 설치사례는 아래 5항목을 **반드시** 포함한다. 순서 변경 불가.

---

### 항목 1: 고객 공정 상황

| 항목 | 내용 |
|------|------|
| 제목 | "공정 상황" / "고객사 개요" |
| 필수 포함 | 업종, 생산 규모, 기존 작업 방식, 공정 특성 |
| 톤 | 객관적·구체적으로, 고객 비밀 유지 범위 내 |

**예시 방향:**  
"○○ 업종, 월 △△개 생산. 기존 수작업 용접, 숙련공 부족으로 납기 불안정."

---

### 항목 2: 요구 사항

| 항목 | 내용 |
|------|------|
| 제목 | "요구 사항" / "해결 필요 사항" |
| 필수 포함 | 품질, 생산성, 안전, 원가 등 고객이 원한 구체 목표 |
| 톤 | 고객 목소리 차용 가능 ("~를 원했다" 형태) |

**예시 방향:**  
"용접 품질 균일화, 작업 시간 30% 단축, 재작업률 5% 이하 유지."

---

### 항목 3: 적용 시스템 구성

| 항목 | 내용 |
|------|------|
| 제목 | "적용 시스템" / "구성" |
| 필수 포함 | 로봇·지그·이송·포지셔너 등 **구성 요소**와 역할 |
| 톤 | 로봇 브랜드·모델은 보조 정보로만, 구성 논리가 주인공 |

**예시 방향:**  
"아크 용접 로봇 1대, 회전 포지셔너 2축, 자재 고정용 지그. 2공정 분할로 사이클 타임 확보."

---

### 항목 4: 현장 대응 포인트

| 항목 | 내용 |
|------|------|
| 제목 | "현장 대응 포인트" / "적용 시 해결한 과제" |
| 필수 포함 | 자재 변동, 치수 공차, 작업 환경 등 **현장 특수성**과 대응 방식 |
| 톤 | 세한의 엔지니어링 역량이 드러나도록 |

**예시 방향:**  
"자재 두께 편차 ±0.5mm → 센서 보정 및 용접 조건 자동 보정으로 대응. 슬래그 발생 구간은 토치 각도 조정."

---

### 항목 5: 결과 및 안정화 내용

| 항목 | 내용 |
|------|------|
| 제목 | "결과" / "도입 효과" |
| 필수 포함 | 품질·생산성·안정화 수치, 양산 전환 시기, 유지보수 현황 등 |
| 톤 | 과장 없이, 수치·기간이 있으면 구체적으로 |

**예시 방향:**  
"시운전 2주 후 양산 전환. 재작업률 8% → 2% 감소. 작업자 1명이 로봇 2대 관리 가능."

---

## 4. 사진·영상 명령어

### 사진

| 원칙 | 내용 |
|------|------|
| 선별 기준 | "많이"보다 **공정 이해가 되는** 것 |
| 포함 권장 | 전체 라인, 지그·치구, 용접 비드, 작업자와의 연동 장면 |
| 배제 | 의미 불명 각도, 과도한 후처리 |

### 영상

| 원칙 | 내용 |
|------|------|
| 목적 | 홍보 영상이 아니라 **현장 동작 확인용** |
| 길이 | 짧을수록 좋음 (30초~1분) |
| 포함 | 실제 용접·이송 동작, 사이클 흐름 |

---

## 5. 설치사례 목록 페이지

| 항목 | 명령어 |
|------|--------|
| 필터/분류 | 공정별(용접/디버링/핸들링/팔레타이징), 업종별 선택 가능 |
| 카드 구성 | 썸네일 + 업종/공정 + 1줄 요약 |
| 정렬 | 최신순 또는 공정별 그룹화 |

---

## 6. 템플릿 체크리스트 (작성 시)

- [ ] 5항목 모두 포함
- [ ] 로봇 브랜드·모델이 주인공이 아님
- [ ] 현장 대응 포인트가 구체적으로 서술됨
- [ ] 결과에 수치·기간이 포함됨 (가능한 경우)
- [ ] 사진은 공정 이해에 도움되는 것만 선별
- [ ] 영상은 현장 동작 확인용으로 사용
