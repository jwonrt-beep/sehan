## 2. 페이지별 상세 섹션 구성안

### 메인 (index.html)
- 메인 비주얼 (슬로건 + CTA 2개)
- 회사 한줄 소개
- 핵심 사업영역 3~5개 요약
- 제품 및 시스템 요약
- 설계→시뮬레이션→설치→관리 프로세스
- 스마트팩토리 역량 요약
- 인증 및 파트너 로고
- 연혁/핵심 숫자 요약
- 사업장 및 문의 유도

### 회사소개 (company.html)
- 회사 개요
- 회사 연혁 (타임라인)
- 대표자 소개
- 경영이념 및 비전
- 조직도
- 임직원 및 경영진 소개
- 인재상/행동양식
- 사업실적 및 제휴 개요

### 사업영역 (business.html)
- 로봇 용접 자동화 / 자동 이송·적재 / 조립용 로봇 / 시뮬레이션 / 스마트팩토리 / 생산관리 연계 — 카드형

### 제품 및 시스템 (products.html)
- 용접 로봇 시스템 / 포지셔너·외부축 / 대형·멀티 로봇 / 모니터링·생산관리 — 카탈로그형

### 스마트팩토리 (smartfactory.html)
- 사업개요 / 모니터링 / ARC / 생산관리

### 구축사례·시뮬레이션 (cases.html)
- 썸네일 리스트 + 상세 — 1축 동기제어, 전선 트레이, 2대 동시제어·용접부스, Jig 2개·SPOT

### 인증 및 파트너 (certifications.html)
- Panasonic 인증대리점 / 경영혁신형 중소기업 / ISO9001 / 표창장 / 무역업·사업자 / 호서대 가족기업 — 갤러리형

### 사업장 (locations.html)
- 시흥본사·시화공장 / 천안본사 — 주소·연락처·지도·사진 (수정 용이 구조)

### 문의 (contact.html)
- 폼: 회사명, 담당자, 연락처, 이메일, 문의분야, 문의내용

## 3. 메인 페이지 와이어프레임(텍스트)

```

[Hero]
  산업용 로봇 자동화 시스템 전문기업
  설계부터 시뮬레이션, 설치 및 관리까지 통합 대응
  [회사소개 보기] [문의하기]

[한줄 소개]
  ONE STOP TOTAL SOLUTION — 산업용 로봇 자동화 시스템의 제작·공급 기업

[사업영역 요약] 5~6 카드
[제품·시스템 요약] 블록
[프로세스] 설계 → 시뮬레이션 → 설치 → 관리
[스마트팩토리] 요약
[인증·파트너] 로고/뱃지
[연혁/숫자] 타임라인 또는 키넘버
[사업장·문의 유도]

[Footer: 회사명·대표·주소·전화·팩스·이메일·메뉴·저작권]
```

## 4. 디자인 시스템 방향

- **메인 컬러**: 네이비(#0f172a, #1e293b), 블루(#1e3a5f, #2563eb 계열 보조), 화이트(#fff), 그레이(#64748b, #94a3b8, #f1f5f9)
- **액센트**: 블루 계열로 통일 (오렌지 제거), 신뢰감 유지
- **타이포**: Noto Sans KR, 제목용 세리프 또는 산세리프 하나
- **카드/테이블**: 여백 충분, border 연한 그레이, 그림자 절제
- **애니메이션**: 최소화, 호버 시 미세한 변화만
- **반응형**: 모바일에서 네비게이션 토글, 카드 1~2열
