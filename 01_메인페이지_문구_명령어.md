# 세한메카트로닉스 메인페이지 문구 명령어

> 로봇 용접 자동화 수주용 영업 사이트의 메인 페이지에 적용할 문구·구조 명령어

---

## 1. 메인 페이지 목적

- **보기 좋은 소개**가 아니라 **불안을 제거하는 구조**
- 3초 안에 "용접 로봇 자동화 특화 기업"임을 인지 가능하게

---

## 2. 사용 금지 표현

| 금지 | 이유 |
|------|------|
| AI·소프트웨어·플랫폼·데이터 중심 | 제이원로보틱스와 겹침 |
| 스마트·지능형·AI 자동화 | 추상적, 신뢰 불리기 어려움 |
| 기술 나열형 문구 | "보기 좋은 소개"에 해당 |
| 과장된 전국 대응 표현 | 현실적 표현 유지 |

---

## 3. 섹션별 문구 명령어

### 3-1. 히어로 섹션 (최상단)

**목표:** 용접 자동화 전문성 즉시 선언

| 항목 | 명령어 |
|------|--------|
| 메인 헤드라인 | "용접 로봇 자동화, 설계부터 유지보수까지" 또는 이와 동등한 직관적 문구 |
| 서브 카피 | 세한이 **용접 공정**에 특화된 기업임을 1문장으로 |
| CTA | "현장 상담 요청" / "무료 견적 문의" 등 즉시 액션 유도 |

**톤:** 확신 있게, 과장 없이

---

### 3-2. 적용 공정 명확화 섹션

**목표:** "우리 공정에도 적용되는가?" 즉시 확인 가능

| 항목 | 명령어 |
|------|--------|
| 섹션 제목 | "적용 가능한 공정" 또는 "이런 공정에 적용합니다" |
| 항목 구성 | 로봇 용접 / 디버링 / 핸들링·이송 / 팔레타이징 (4개 고정) |
| 각 항목 문구 | 공정명 + 한 줄 설명 (기술 용어 최소화) |

**예시 톤:** "스폿 용접, 아크 용접, 레이저 용접 등 — 자재·두께·형상에 맞춰 적용"

---

### 3-3. 불안 해소 3가지 섹션

**목표:** 고객 핵심 불안 3가지를 각각 해소

| 불안 | 해소용 제목 | 해소용 본문 방향 |
|------|-------------|------------------|
| 용접 품질 | "사람 이상의 품질, 반복 가능" | 파라미터·지그·공정 설계로 일관 품질 |
| 고장·유지보수 | "고장 시 누가, 얼마나 빨리" | 현장 방문 기준, 대응 프로세스 명시 |
| 투자 효과 | "투자 대비 실제 효과" | 작업 시간 절감, 품질 불량 감소 등 구체 수치 가능 시 제시 |

**문구 원칙:** 질문형 제목 → 답변형 본문

---

### 3-4. 설치 사례 노출 섹션

**목표:** "실제로 설치해봤구나" 신뢰 형성

| 항목 | 명령어 |
|------|--------|
| 섹션 제목 | "설치 사례" / "현장 적용 사례" |
| 각 사례 | 업종 또는 공정명 + 1줄 요약 (상세는 별도 페이지) |
| CTA | "더 많은 설치 사례 보기" |

**금지:** 제품 나열형, 로봇 브랜드·모델 강조

---

### 3-5. 세한 수행 방식 섹션

**목표:** "어떻게 진행하는가" 투명하게

| 항목 | 명령어 |
|------|--------|
| 섹션 제목 | "세한의 수행 방식" / "설계부터 현장까지" |
| 흐름 표현 | 설계 → 제작 → 설치 → 시운전 → 교육 → 유지보수 (전 공정 직접 수행) |
| 강조 문구 | "외주 분산 없이, 한 곳에서 끝까지 책임" |

---

### 3-6. 유지보수·교육 강조 섹션

**목표:** "시공 후에도 책임진다" 인지

| 항목 | 명령어 |
|------|--------|
| 유지보수 | 대응 프로세스, 현장 방문 기준, 예방 점검 가능 여부 언급 |
| 교육 | "강의"가 아니라 "현장 운영 가능성" — 작업자가 실제로 운영할 수 있게 |

---

### 3-7. 즉시 상담 유도 섹션

**목표:** 문의 진입 장벽 최소화

| 항목 | 명령어 |
|------|--------|
| 제목 | "현장 상황을 알려주시면, 맞춤 검토합니다" |
| 문구 | 복잡한 설명 없이, **현장 정보를 빠르게 받는** 구조 강조 |
| CTA | 전화 / 이메일 / 간단 문의폼 |

## 5. 세한 강점 반복 노출 포인트

다음 4가지를 메인 페이지 내 적절한 위치에 반복:

1. 용접 품질 안정화 경험
2. 지그·포지셔너·치구 설계 역량
3. 시운전 및 양산 안정화 경험
4. 작업자 교육 및 현장 인수인계
